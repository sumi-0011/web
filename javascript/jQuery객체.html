jQuery 객체

jQuery 객체란?

jQuery의 함수의 리턴값으로 jQuery함수를 이용해서 선택한 엘리먼트들에 대해서 처리할 작업을 프로퍼티로 가지고 있는 객체다.

암시적 반복

jQuery 객체의 가장 중요한 특성은 암시적인 반복을 수행한다는 것이다. DOM 과는 다르게 jQuery 객체의 메소드를 실행하면 선택된 엘리먼트 전체에 대해서 동시에 작업이 처리된다.
암시저 반복은 값을 설정할 때만 동작한다. 값을 가져얼 떼는 선택된 엘리먼트 중 첫번째에 대한 값만을 반복한다. 이에 대한 내용은 아래에서 살피기로한다.
